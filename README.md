# Netflix Data Analytics Dashboard

## Project Overview

This project analyzes Netflix content using Microsoft Excel. The dataset contains information about Netflix movies and TV shows, including content type, ratings, release years, countries, directors, and genres.

The objective of this project is to transform raw Netflix data into meaningful insights using Pivot Tables, Pivot Charts, and an interactive dashboard.

---

## Dataset

Source: Netflix Titles Dataset

File: `data/netflix_titles.csv`

The dataset contains:
- Show ID
- Content Type (Movie / TV Show)
- Title
- Director
- Cast
- Country
- Date Added
- Release Year
- Rating
- Duration
- Genre (Listed In)

---

---

## Analysis Performed

### 1. Content Distribution
Analyzed the number of Movies and TV Shows available on Netflix.

### 2. Content Rating Analysis
Examined the distribution of content ratings such as TV-MA, TV-14, PG, and R.

### 3. Release Year Trends
Analyzed how Netflix content has grown over time based on release year.

### 4. Top Countries
Identified countries contributing the highest number of titles to Netflix.

### 5. Top Directors
Determined directors with the largest number of titles in the dataset.

## Dashboard Features

The dashboard provides visual insights through:
